# 更新记录

## 2.1.0

- 增加面向阅读和归档的 Markdown / TXT 导出
- 支持从当前对话、搜索结果、收录库单卡片或底部批量入口选择导出对象
- 导出前显示选中对话数量、消息数量和未完整保存警告
- 通过浏览器下载 API 打开“另存为”窗口，让用户选择文件名和保存位置
- 导出失败时尝试将内容复制到剪贴板作为兜底
- 将“导出”和 JSON“备份/恢复”在界面与说明中区分开
- 增加导出核心逻辑测试

## 2.0.0

- 将整段快照覆盖改为消息级增量合并，离开页面 DOM 的消息不再被删除
- 增加 v1 本地记录与备份的自动迁移
- 增加“扫描并完整保存”流程：定位顶部、分段向下、逐批落盘、底部确认和顺序校验
- 增加扫描 HUD、暂停、继续、停止并保留以及原阅读位置恢复
- 增加收录库、完整状态筛选和搜索可信度提示
- 搜索无结果时提示仍未完整收录的对话数量
- 增加本地存档查看与原对话定位入口
- 增加消息区段合并、缺口检测、流式回复合并和迁移测试

## 1.1.0

- 增加本地索引备份与恢复
- 增加长期记忆说明

## 1.0.0

- 首个可用版本
- 实时读取当前页面中的 ChatGPT 消息并提供本地关键词搜索
